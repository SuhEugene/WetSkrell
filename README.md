# WetSkrell
Моя попытка сделать интересный мем красивым


## Кавочевокуда

Установка зависимостей:
```bash
yarn install
```

Запуск сервера разработки:
```bash
yarn dev
```

Билд для прода:
```bash
yarn build
```

Локально посмотреть билд прода:
```bash
yarn preview
```
